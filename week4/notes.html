Chapter 8, 12, and 15

-to create a form in html use " < 'form'> < '/form'>"
-Click here to see an example of a search bar
-document.forms that returns an HTML collection of all the forms in the document in the order they appear in the markup.
- "const form = document.forms[0];" returns the first form
-We can also use .(name attribute) to get the desired form
-You need to make sure the name is unique if you use this method
-You can also use "form.elements" to return all of the elements in the form
-You can retrieve specific elements by using "form.(element name)"
-The form.submit() method will submit the form automatically
-The form.reset() method will reset all the form controls back to their initial values specified in the HTML
-The focus event occurs when an element is focused onThe focus event occurs when an element is focused on -The blur event occurs when the user moves the focus away from the form element.
-The change event occurs when the user moves the focus away from the form element after changing it.
-Some common types of form control are:

- < 'input'> fields, including text, passwords, check boxes, radio buttons, and file -uploads
- < 'select'> menus for drop-down lists of options
- < 'textarea'> elements for longer text entry
- < 'button'> elements for submitting and resetting forms

-event.preventDefault() method to stop the form from being submitted.
-The default type of input field is text
-input type='password' is used to enter passwords
-Check boxes are created using input fields with type='checkbox'
-Radio buttons are created using input fields with type='radio'
-Hidden fields can be created using input fields with type='hidden'
-A file input field can be created using input fields with type='file'
-There are lots of new input types included in HTML5, such as number, tel and color
-A < 'textarea'> element is used to enter long pieces of text over multiple lines such as a comment or blog post.
-The other type is simply button. This doesn’t need to be inside a form element and has no default behavior
-There is also a type of 'menu' that can be combined with, and tags to create a dropdown menu when it’s clicked on
-Form validation is the process of checking whether a user has entered the information into a form correctly
-Polymorphism means various objects can share the same method
- Inheritance means we can take an object that already exists and inherit all its properties and methods use a class to define a blueprint for an object
